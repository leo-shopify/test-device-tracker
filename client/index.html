Devices and accessories

Ground rules

We have a large set of mobile devices that Shopify employees can use for testing. These are shared devices, so before you get started, please read through some shared device ground rules:

  1. OS version must not be updated. Each device has been labelled with its name and OS. As we have to test various versions of an OS, it's crucial that we keep a stock of older devices and OS's that represent our real-world users. If you need a version to test that we don't already have, please contact one of the device maintainers.

  2. DO NOT add any personal data, apps, accounts, etc. to phones. Phones belong to Shopify and use a common @shopify account. Specific apps can be added as needed for comparison, inspiration, etc, but these should not be tied to any personal emails. Also remember that these devices may be restored to default settings at any time, so don't get too attached to anything.
